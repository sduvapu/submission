VI Editor

On Terminal

This is where you will learn about VI Commands. VI stands for Visual Instrument. The VI editor is the text editor in the Linux family. Using the VI editor allows you to edit documents and files via Terminal.

vi filename opens the file in the vi editor/command mode
There are many ways to insert text into the file.
Enter “I” to insert text (It should show up as -- INSERT -- on the bottom)
You can add whatever text you want such as “Hello, my name is Susmitha”
a will enter text after the cursor, whereas I will enter text at the cursor. Both will enter the insert text mode.
o will open a new line, and then enter the insert text mode.
Press esc to exit the insert mode

There are many ways to delete text from the file.
If you want to delete one character, you can use x. x will delete the character at the cursor.
dd will delete a whole line (the line where the cursor is at).
D will delete all the content on a particular line after the cursor.

There are many ways to exit the VI editor.
So save and exit, either enter “:wq” or “:x”
:wq will always change the timestamp (always touches the file) regardless of what (if any) changes were made
:x will only touch the file if there were changes made
:w will only save your file, but not exit the editor :q will quit the editor and not save the file. This mean any changes you have made will be lost.
You can move the cursor using the arrow keys, or use k to move up, j to move down, h to move left, and l to move right.

Table about VI Commnads

Command Purpose
I to enter text at the cursor - enters insert mode
a to enter text after the cursor - enters insert mode
o open a new line - enters insert mode
ESC to escape insert text mode
dd deletes a line
x deletes character at the cursor
D deletes content on line after cursor
C deletes content on line after cursor and then enters insert mode
j moves the cursor up
k moves the cursor down
h moves the cursor left
l moves the cursor right
:x saves the file only if there are any changes and quits
:wq saves the files and quits
:q quits the file without saving
:w saves the file, but remains in editor mode
